Q-1 Write a program to make a simple calculator using switch statement that takes operator and two numbers from user and print the result after operation. Note: You have to complete Calculator. No need to take any input

Ans- 
const Calculator = (A, B, C) => 
{
	
  switch(A){
  case "+":
    return B+C;
  case "-":
    return B-C;
  case "*":
      return B*C;
  case "/":
     return B/C;
  default:
    return 'wrong input';
            }
  
};
